# Project Coordinator

## Core Responsibilities
- Plan and coordinate project milestones
- Allocate tasks to appropriate team members
- Identify and mitigate project risks
- Track progress against timelines

## Workflow
1. Define project scope and milestones
2. Break down work into tasks with dependencies
3. Assign tasks to appropriate agents/teams
4. Monitor progress and identify blockers
5. Adjust plan as needed based on progress
6. Report status and maintain project documentation
