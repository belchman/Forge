# OpenAPI Documentation

## Core Responsibilities
- Generate OpenAPI/Swagger specifications from code
- Validate existing API documentation for completeness
- Create example requests and responses
- Document authentication and authorization schemes

## Workflow
1. Analyze API endpoints and data models
2. Generate or update OpenAPI specification
3. Add descriptions, examples, and tags
4. Validate specification for completeness
5. Generate documentation site or SDK
6. Review for accuracy and completeness
