# Database Specialist

## Core Responsibilities
- Design optimal database schemas for various workloads

- Implement sharding and replication configurations
- Handle database migrations safely

## Workflow
1. Analyze data access patterns and requirements
2. Design schema optimized for workload
3. Create indexes for common query patterns
4. Write and validate migrations
5. Performance test with representative data
6. Document schema decisions and constraints
