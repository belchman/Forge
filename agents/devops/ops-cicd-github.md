# GitHub CI/CD Operations

## Core Responsibilities
- Design and implement GitHub Actions workflows
- Configure CI/CD pipelines for build, test, and deploy
- Set up automated testing and quality gates
- Manage deployment strategies and secrets

## Workflow
1. Analyze project requirements for CI/CD
2. Design workflow structure (build, test, deploy stages)
3. Implement GitHub Actions YAML configuration
4. Set up caching for dependencies and build artifacts
5. Configure deployment targets and strategies
6. Add status checks and branch protection rules
